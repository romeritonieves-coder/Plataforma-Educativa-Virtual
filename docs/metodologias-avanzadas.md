# Metodologías Avanzadas

Este documento describe metodologías avanzadas de desarrollo de software.

## Scrum

Scrum es un marco de trabajo ágil para gestionar proyectos complejos.

### Ceremonias
- **Sprint Planning**: Planificación del sprint
- **Daily Standup**: Reunión diaria de seguimiento
- **Sprint Review**: Revisión del sprint
- **Retrospectiva**: Mejora continua

## Kanban

Kanban es un método para gestionar el flujo de trabajo.

### Principios
- Visualizar el flujo de trabajo
- Limitar el trabajo en progreso
- Gestionar el flujo
- Hacer explícitas las políticas
- Implementar retroalimentación
- Mejorar experimentalmente

## Extreme Programming (XP)

XP es una metodología ágil para desarrollo de software.

### Prácticas
- Programación en parejas
- Desarrollo basado en pruebas
- Integración continua
- Refactorización
- Diseño simple
